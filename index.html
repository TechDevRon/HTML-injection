HTML INJECTION/XSS

Infomation on HTML injection.

What is HTML Injection?
HTML Injection also known as Cross Site Scripting.
It is a security vulnerability that allows an attacker to inject HTML code into web pages that are viewed by other users.

What can an attacker with HTML Injection?
An attacker can use HTML injection to, bypass login pages, steal data, and phishing on an offical domain
    
How could you find a HTML injection/Xss vulnerability and project against it?
You can look at how javascript code process the input and find a loop hole in the code.
One way to stop this is by using a obstruciton.
